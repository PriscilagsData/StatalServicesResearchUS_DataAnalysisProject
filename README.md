# Statal Services Research (US) - Data Analysis
Power BI

2026 study of contract bidding opportunities for an investment advisory company in confidential private sector.

Historical contract data is analyzed, time trends, amounts, frequently contracted companies, and public entities requesting services.
Opportunities are examined based on expiration dates, categorized by urgency, amounts, location, and publication dates.

Data from csv, Excel, and Snowflake. Mockup design, KPI definition, data visualization. Power BI tailored to client's needs to identify trends and valuable insights.

Some data was modified for reasons of corporate confidentiality.
